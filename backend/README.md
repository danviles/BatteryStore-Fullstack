# Backend
E08

Para usar el backend se debe seguir las siguientes instrucciones

Instalamos las dependencias del proyecto
```
npm install
```

Para ejecutarlo en modo desarrollo
```
npm run dev
```

Para iniciar el backend
```
npm run start
```
